🎮 What Is This?

Pop A Lock is a daily puzzle game that applies Wordle's guess-and-feedback loop to numeric combination codes. Instead of guessing a word, you're guessing a secret 4-digit code. Each guess returns two pieces of feedback: how many digits are correct and in the right position, and how many are correct but in the wrong position — the same mechanic as the classic Mastermind board game.

The daily seed means every player gets the same puzzle on a given day, enabling the same kind of informal social comparison that made Wordle popular — "I got it in 3, how about you?" It's a satisfying 2–5 minute daily puzzle, different in feel from word-based games like Spelling Bee and Word Mastermind but fitting naturally alongside them as part of a daily puzzle routine.

📖 How to Play
  • Enter a 4-digit guess using the number pad
  • Submit to receive feedback: digits correct in position (🟩) and digits correct but misplaced (🟨)
  • Use the feedback to narrow down the correct combination
  • Solve it in as few guesses as possible — fewer is better
  • The puzzle resets at midnight to a new daily code

🏆 Statistics

The game tracks your solve streak and guess distribution in localStorage — similar to Wordle's stats panel. Because all players solve the same daily puzzle, the number of guesses used is a natural informal comparison metric.

A shared leaderboard showing each player's guess count for today's puzzle would fit this game extremely well. The completion event could POST to the hub's score API with the guess count as the score. Document that work here if undertaken.
