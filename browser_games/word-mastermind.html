Word Mastermind

// Word Puzzle · Singleplayer · Node.js / Static · Self-Hosted · Tailscale + Authentik forward_auth · MIT //

A Wordle-style word guessing game with a Mastermind twist. Guess a hidden 5-letter word in 6 attempts. After each guess, colour-coded feedback shows which letters are correct and in position (green), present but misplaced (yellow), or absent (grey). Clean, minimal, and fully self-hostable.

📁 Step 1 — Clone Repository

The game is a Node.js project that serves static files. Clone and install:

cd D:\Server\BrowserGames
git clone https://github.com/clupasq/word-mastermind.git word-mastermind
cd word-mastermind
npm install
💡
After npm install, check whether there is a dist/ or public/ folder. If the project uses a bundler, run npm run build first. The entry point should be index.html at the build output root.

🔤 What Is This?

Word Mastermind is a Wordle clone built on Mastermind's colour-coded feedback principle. You have 6 attempts to guess a secret 5-letter word. After each guess, each letter tile turns green (correct letter, correct position), yellow (correct letter, wrong position), or grey (letter not in the word).

The Mastermind framing emphasises the deductive logic angle — each guess is a hypothesis, and the feedback is your experimental result. The game is clean, minimal, and requires no account or backend to play.

⌨️ Controls

Type letters with your keyboard and press Enter to submit a guess. Backspace removes the last letter. An on-screen keyboard is available for touchscreen play.
