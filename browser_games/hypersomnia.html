🐳 Optional — Self-Host a Dedicated Server

If you want private LAN matches without using community servers, Hypersomnia ships a dedicated server Docker image.

hypersomnia-server:
  image: teamhypersomnia/hypersomnia:latest
  ports:
    - "8412:8412/udp"
  restart: unless-stopped
  environment:
    - SERVER_NAME=ixo-private

Players on the browser client can then connect to <your-tailscale-ip>:8412 from the in-game server browser.

⚠️
The dedicated server image requires a publicly accessible UDP port for external players. For LAN/Tailscale-only play, the Tailscale IP is sufficient — no port forwarding needed.

🔫 What Is This?

Hypersomnia is a free, open-source competitive top-down shooter developed by Team Hypersomnia. The core game mode is a Counter-Strike-style bomb defusal: two teams — terrorists and counter-terrorists — play round-robin matches with a single life per round. One team plants a bomb; the other defuses it or eliminates everyone.

The top-down perspective and movement speed give the game a feel closer to Hotline Miami — engagements are fast, spatial awareness matters, and a single bullet can end your round. The map design rewards positioning and information.

What makes Hypersomnia special among FOSS games is its full WebAssembly browser build. Visiting play.hypersomnia.io loads the complete game client, connects to community servers, and lets you play immediately without installing anything. The development team recently fixed cross-origin isolation and iframe embedding, meaning it can be run inside the Browser Games hub frame.
