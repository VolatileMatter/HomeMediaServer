Generative Trees

// Generative Art · Singleplayer · Static Files (p5.js) · Tailscale + Authentik forward_auth //

A fractal tree generator built with p5.js. Recursive branching algorithms produce stylised trees that grow outward from a trunk, with randomised angles, lengths, and leaf densities. Each tree is unique and grows in real-time — watching the branches emerge is half the experience.

🎮 What Is This?

Generative Trees uses recursive branching to produce stylised tree illustrations in real-time. Starting from a single trunk, the algorithm splits into two branches at a random angle, then each branch splits again, and so on until branches become leaves. The randomness at each split means no two trees are ever the same.

The p5.js framework makes this kind of sketch approachable and the result is visible growing in real-time — branches emerge from the centre outward in a satisfying sequence. It is one of the canonical examples of L-system-style procedural generation.

This is a calming, visual toy. You can click to regenerate a new tree, or watch a sequence of trees grow and settle.

📖 How to Use
  • The tree begins growing as soon as the page loads
  • Click the canvas to generate a new tree with fresh randomisation
  • Watch branches emerge from the trunk outward — the growth animation is part of the experience
  • The final shape settles in a few seconds and remains static until you regenerate
