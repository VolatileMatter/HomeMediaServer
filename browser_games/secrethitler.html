⚠️ Architecture Note
⚠️
Secret Hitler requires a persistent Node.js server for game state management and role assignment. It is not a static file game. Runs as a single Docker container.
🐳 Step 1 — Docker Compose

The repo ships with a Dockerfile. Build and run it:

cd ~/services
git clone https://github.com/brdly/Secret-Hitler.git secrethitler
cd secrethitler

Create ~/services/secrethitler/docker-compose.yml:

services:
  secrethitler:
    build: .
    container_name: secrethitler
    restart: unless-stopped
    ports:
      - "8860:8080"
docker compose up -d --build
🌐 Step 2 — Caddy Configuration

Add to :

secrethitler.{$DOMAIN} {
  forward_auth  {
    uri /outpost.goauthentik.io/auth/caddy
    copy_headers X-authentik-username X-authentik-groups
  }
  reverse_proxy localhost:8860
}

🎮 What Is This?

Secret Hitler is a social deduction game for 5–10 players, originally a physical card game available under a Creative Commons license. Players are secretly assigned roles: the majority are Liberals, a smaller group are Fascists, and one player is secretly Hitler. Fascists know each other's identities; Liberals know nothing.

Each round, a President and Chancellor are nominated. They draw and enact policy tiles — but the deck is stacked with more Fascist policies than Liberal ones, creating pressure and distrust even among honest players. Fascists win by enacting 6 Fascist policies or getting Hitler elected Chancellor after 3 Fascist policies are in place. Liberals win by enacting 5 Liberal policies or assassinating Hitler.

The game produces extraordinary moments of accusation, misdirection, and paranoia. It's one of the best social games for a friend group that enjoys argument and bluffing. A single game takes 20–40 minutes.

📖 How to Play
  • One player creates a room and shares the join link with 4–9 others
  • Once all players join, the host starts the game — roles are assigned secretly
  • Each round: nominate a President → President nominates a Chancellor → players vote Ja/Nein
  • If the government is confirmed: President draws 3 policy tiles, discards 1, passes 2 to Chancellor who discards 1 and enacts 1
  • Special powers unlock for the Fascist team as Fascist policies accumulate
  • Liberals: track who votes how, use deduction, trust nobody claiming to be Liberal too loudly
  • Fascists: blend in, misdirect, protect Hitler
⚠️ Known Limitations
IssueDetail
Minimum 5 playersThe role distribution requires at least 5. Fewer players cannot start a game.
No rejoin after disconnectIf a player disconnects mid-game they may not be able to rejoin the same game. The room persists but the disconnected player's seat may be lost.
No spectatorsPlayers who join late cannot observe an in-progress game.
👥 Secret Hitler is strictly a multiplayer game — it requires 5–10 players and cannot be played solo or with fewer than 5.
