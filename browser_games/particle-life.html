🎮 What Is This?

Particle Life is an implementation of the "particle life" algorithm, a type of artificial life simulation. Hundreds or thousands of coloured particles move around a canvas, each attracted to or repelled by particles of other colours according to a matrix of rules you control. Despite the simplicity of the rules, the resulting behaviour is remarkably organic — particles form stable clusters, hunting chains, and pulsing blobs reminiscent of microscopic life.

The original concept draws from work on artificial chemistry and emergence. Hunar Ahmad's browser implementation makes the rule matrix directly editable via sliders, letting you sculpt the emergent behaviour in real-time. Change a single attraction value and the entire ecosystem responds.

There is no goal. It is a scientific toy — part simulation, part art, part experiment. Watching a random configuration spontaneously self-organise into stable, repeating patterns is genuinely mesmerising.

📖 How to Play
  • The simulation starts automatically with a random rule matrix
  • Use the sliders to set attraction/repulsion between each pair of particle colours (positive = attract, negative = repel)
  • Hit Randomise to get a new rule matrix and watch behaviour shift
  • Adjust particle count and simulation speed with the top controls
  • Look for stable configurations — some rule sets produce remarkably persistent structures
Controls
Sliders Set attraction/repulsion between colour pairs Randomise Generate a new random rule matrix Reset Scatter particles to random positions Count slider Adjust total particle count (more = slower)
⚠️ Known Limitations
IssueDetail
PerformanceParticle count is CPU-bound. 500 particles runs smoothly on any modern machine; 2000+ may slow older hardware. The simulation is single-threaded JavaScript.
MobileThe simulation renders fine on mobile, but the control sliders are small and difficult to use on a phone. Tablet or desktop recommended.
No saveThere is no way to save a rule configuration. If you find a pattern you love, note down the slider values manually.
