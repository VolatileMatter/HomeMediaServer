⚠️ Architecture Note — Legacy Repository
⚠️
This is the legacy OpenSuspect repository (opensuspect-legacy). The project was ambitious but development stalled. The legacy repo represents the most complete playable state. Check the README carefully — the HTML5 browser export and self-hosted server setup may require specific Godot version tooling to rebuild from source if the prebuilt binaries are not included.
📁 Step 1 — Clone & Check Builds
cd ~/services
git clone https://github.com/opensuspect/opensuspect-legacy.git opensuspect
cd opensuspect

Check the repo for prebuilt exports — look for a export/, build/, or releases/ directory, or check the GitHub Releases tab for downloadable binaries (server binary + HTML5 client export).

💡
If no prebuilt exports exist, you will need Godot 3.x to export the HTML5 client and compile the server binary. Check the repo README for the exact Godot version required — this project predates Godot 4.
🐳 Step 2 — Server Setup

If a server binary is available, run it directly or wrap it in Docker:

services:
  opensuspect:
    build: .
    container_name: opensuspect
    restart: unless-stopped
    ports:
      - "8710:8080"    # HTTP for HTML5 client
      - "4321:4321/udp" # Game server UDP port (check README)
docker compose up -d --build
🌐 Step 3 — Caddy Configuration
opensuspect.{$DOMAIN} {
  forward_auth  {
    uri /outpost.goauthentik.io/auth/caddy
    copy_headers X-authentik-username X-authentik-groups
  }
  reverse_proxy localhost:8710
}

🎮 What Is This?

OpenSuspect is an open-source attempt to recreate the Among Us formula: a group of players aboard a ship, divided secretly into Crewmates and Impostors. Crewmates complete tasks around the ship to win; Impostors sabotage systems and eliminate Crewmates without being caught. When a body is found or an emergency meeting is called, everyone discusses and votes on who they think the Impostor is.

The game is built with Godot and exports to HTML5, making it browser-playable without a native install. The server component handles matchmaking and game state. Being open source and self-hosted means a private server with no strangers, no account requirements, and no dependency on Innersloth's servers.

The legacy codebase represents the most complete playable state of the project. Feature completeness varies — check the README for what mechanics are implemented in this version versus the full Among Us feature set.

SharedArrayBufferGodot HTML5 exports require Cross-Origin-Opener-Policy and Cross-Origin-Embedder-Policy headers for threading. The server must set these — check if Docker/Caddy handles them correctly.
Minimum 4 playersSocial deduction requires a minimum player count to function meaningfully.
👥 OpenSuspect requires 4–10 players. All players use the HTML5 browser client — no native install needed.
