📁 Step 1 — Clone & Build

Alphabet Gacha is a Vite/Phaser project requiring a build step.

cd D:\Server\BrowserGames
git clone https://github.com/AccelByte/extend-alphabet-gacha-client.git alphabet-gacha
cd alphabet-gacha
npm install
npm run build

Built static files land in dist/. Serve that folder as the game directory.

🎮 What Is This?

Alphabet Gacha is an idle clicker and gacha hybrid developed by AccelByte as a sample/demo project for their game backend SDK. The gameplay loop is a distilled version of what makes idle gacha games compelling: click to generate virtual currency, spend currency on pulls to receive random alphabet symbols (A through Z), and work toward completing the full set.

The collector psychology driving full-set completion is the same mechanism used in trading card games and gacha titles — even a simple alphabet set creates genuine satisfaction when the rare letters finally drop. Phaser.js provides smooth canvas rendering for the pull animations and idle currency generation.

⚠️
AccelByte SDK note: This project was designed to demonstrate AccelByte's backend services. Depending on the version, it may require AccelByte API credentials to function fully. If it fails to load or pulls don't work, check the browser console for API errors — you may need to fork the project and stub out the AccelByte calls to make it standalone.
📖 How to Play
  • Click the main area to generate virtual currency (idle games also accumulate currency over time)
  • Spend currency on pulls to receive a random alphabet symbol
  • Track which letters you've collected — rarer letters (Q, X, Z) appear less frequently
  • Goal: collect all 26 letters of the alphabet
Controls
Click / tap Generate currency (idle clicker) Pull button Spend currency for a random alphabet symbol
