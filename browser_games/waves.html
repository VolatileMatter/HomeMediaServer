🎮 What Is This?

Waves renders a continuous, animated 3D wave surface using WebGL via Three.js. Multiple sine waves are layered and shifted over time, creating interference patterns that produce flowing, organic-feeling geometry. The colour palette and lighting shift alongside the motion.

Created by Klevron, this is part of a broader series of Three.js visual experiments focused on ambient, non-interactive art. The aesthetic is calm and hypnotic — the kind of thing you might leave open on a second monitor or use as a screensaver substitute.

There are no controls and no goal. Some versions expose parameter controls (frequency, amplitude, colour) — check the specific build for available options.

📖 How to Use
  • Open the page — the animation starts automatically
  • Full-screen the browser tab for maximum effect (F11)
  • Some builds have a GUI panel (dat.GUI) for adjusting parameters — look for a small panel in the corner
  • Leave it running as ambient background art

⚠️ Known Limitations
IssueDetail

👥 Multiplayer

This is a singleplayer experience. There is no networked, local co-op, or async multiplayer mode.

The game runs entirely client-side with no server state, making multiplayer non-trivial to add. If a multiplayer variant is ever introduced (e.g. a shared canvas mode), document it here and replace this pane with the full multiplayer template from browser-game-template.html.
