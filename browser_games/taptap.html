⚠️ Relationship to BrowserQuest
💡
Tap Tap Adventure is a fork of BrowserQuest with expanded content. If you already have BrowserQuest running, consider whether you want both or just Tap Tap Adventure. They are separate deployments on separate subdomains — they do not share a world or player state.
🐳 Step 1 — Clone & Build
cd ~/services
git clone https://github.com/Tach-Yon/Tap-Tap-Adventure.git taptap
cd taptap

Create ~/services/taptap/docker-compose.yml:

services:
  taptap:
    build: .
    container_name: taptap
    restart: unless-stopped
    ports:
      - "8810:8000"   # WebSocket game server
      - "8811:80"     # Static client files
docker compose up -d --build
⚠️
Like BrowserQuest, Tap Tap Adventure targets an older Node.js version. If the build fails, check the repo README for the required version or try with --legacy-peer-deps during npm install.
🌐 Step 2 — Caddy Configuration

Add to :

taptap.{$DOMAIN} {
  forward_auth  {
    uri /outpost.goauthentik.io/auth/caddy
    copy_headers X-authentik-username X-authentik-groups
  }

  handle /ws* {
    reverse_proxy localhost:8810
  }
  handle {
    reverse_proxy localhost:8811
  }
}

🎮 What Is This?

Tap Tap Adventure is a community-developed expansion of Mozilla's BrowserQuest. Where BrowserQuest is a 15–30 minute linear adventure that ends with a single boss fight, Tap Tap Adventure substantially extends the content: additional map areas, more enemy types, expanded item sets, character classes, and quests that give the world more structure and replayability.

The core experience is identical to BrowserQuest — a top-down HTML5 RPG where multiple players share a world in real time, click to move and attack, and collect gear. The expansion simply offers significantly more of it. Players who have completed BrowserQuest will find new areas to explore and a longer progression arc.

Running both BrowserQuest and Tap Tap Adventure is viable — they are independent deployments on separate subdomains with no shared state.

📖 How to Play
  • Enter a name to spawn into the starting area
  • Click to move and attack enemies
  • Collect gear dropped by enemies to upgrade equipment
  • Talk to NPCs for quests and direction
  • Explore new zones beyond the original BrowserQuest map
  • Other players share your world in real time — cooperate or go solo
