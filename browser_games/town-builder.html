🎮 What Is This?

Town Builder generates medieval settlement maps using a wave function collapse-adjacent algorithm. Rather than simply placing buildings randomly, it models organic urban growth: roads extend outward from a centre, plots subdivide along the road network, and buildings fill the plots in a way that feels like a town that grew over centuries rather than one planned all at once.

Oskar Stålberg is a Swedish game developer best known for Bad North and Townscaper. Town Builder is a browser prototype that preceded Townscaper and remains freely available as a standalone web toy. The underlying algorithm — where local constraints propagate to produce globally coherent structures — has become influential in the procedural generation community.

The output is a top-down map that could plausibly serve as inspiration for a tabletop RPG setting, a game level, or simply something pleasant to look at.

📖 How to Use
  • A town generates automatically when the page loads
  • Click the canvas or a regenerate button to produce a new layout
  • Some versions allow clicking on individual plots to cycle their building type
  • Zoom in with the browser (Ctrl +) to appreciate the fine detail in the map linework

👥 Multiplayer

This is a singleplayer experience. There is no networked, local co-op, or async multiplayer mode.

The game runs entirely client-side with no server state, making multiplayer non-trivial to add. If a multiplayer variant is ever introduced (e.g. a shared canvas mode), document it here and replace this pane with the full multiplayer template from browser-game-template.html.
