🎮 What Is This?

Bird Generator produces stylised bird illustrations using procedural SVG path construction. Each bird is assembled from a set of parametric components — body ellipse, wing angle and shape, tail fan, beak type, eye placement, and colour palette — all randomised within aesthetically pleasing bounds.

The result is a collection of charming, folk-art-adjacent bird portraits that look like they could belong in a field guide designed by a whimsical illustrator. The style is flat, graphic, and clean.

Created by Michael Kolesidis, a developer with a focus on creative coding and generative art. The project is simple in concept but produces surprisingly varied and likeable output. It is one of the most "shareable" toys in the collection — the birds are immediately relatable and often amusing.

📖 How to Use
  • A bird is generated immediately on page load
  • Click the Generate button (or the canvas) to produce a new bird
  • SVG output means the bird scales to any size without quality loss — zoom in freely
  • Right-click the SVG to save it, or take a screenshot

Click / Generate Produce a new random bird

This game has no win condition or score. It is a generative toy / sandbox — the experience is in the exploration, not the outcome. A leaderboard would not be meaningful here.

👥 Multiplayer

This is a singleplayer experience. There is no networked, local co-op, or async multiplayer mode.

The game runs entirely client-side with no server state, making multiplayer non-trivial to add. If a multiplayer variant is ever introduced (e.g. a shared canvas mode), document it here and replace this pane with the full multiplayer template from browser-game-template.html.
