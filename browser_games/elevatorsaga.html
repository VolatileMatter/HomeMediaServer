🎮 What Is This?

Elevator Saga is a programming challenge game where you write JavaScript to control the elevators in a simulated building. Each challenge presents a scenario: N floors, M elevators, K passengers per minute arriving, and a goal — transport a certain number of people within a time limit, or minimise average waiting time.

Your code is executed against a live physics simulation. You can watch the elevators move in real time as your algorithm runs. Bugs in your logic manifest visually — elevators sitting idle while passengers wait, or all elevators bunching on the same floor. Tuning the algorithm to pass tighter challenges requires thinking about scheduling, queuing theory, and edge cases.

There are 18 challenges of increasing difficulty. The game is a classic of the browser-based coding puzzle genre, alongside Untrusted, and is well-suited to developers who enjoy optimisation problems.

👩‍💻 Best for: developers 🧩 Puzzle depth: medium-high 📚 Teaches: algorithms, event-driven code

📖 How to Play
  • Read the challenge requirements — number of floors, elevators, and the transport goal
  • Write JavaScript in the code editor to implement init and update functions
  • Your code receives elevator and floor objects with methods like goToFloor(), on("idle"), etc.
  • Click Apply to run your code — watch the simulation and see if the challenge is met
  • Adjust and re-apply until the goal is reached within the time limit
  • Complete the challenge to unlock the next one
Key API Methods
elevator.goToFloor(n) Send elevator to floor n elevator.on("idle", fn) Callback when elevator has no destinations floor.on("up_button_pressed") Callback when someone presses the up button elevator.currentFloor() Returns the elevator's current floor number elevator.loadFactor() Returns how full the elevator is (0–1)
