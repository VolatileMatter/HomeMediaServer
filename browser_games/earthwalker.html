⚠️ Architecture Note — Google API Key Required
⚠️
Earthwalker uses the Google Street View Static API and Google Maps JavaScript API to load panoramas and display the guess map. A Google Cloud API key with both APIs enabled is required. This will incur Google API costs if usage exceeds the free tier. For a small friend group playing occasionally, costs are typically minimal but not zero.
🔑 Step 1 — Google API Key
  1. Go to console.cloud.google.com
  2. Create a new project (or use an existing one)
  3. Enable: Maps JavaScript API and Street View Static API
  4. Create an API key under Credentials; restrict it to your server's IP or domain for safety
  5. Set a billing alert — Street View API has a free tier (~$200/month credit) but charges beyond that
🐳 Step 2 — Docker Compose

Create ~/services/earthwalker/docker-compose.yml:

services:
  earthwalker:
    image: ghcr.io/daehli/earthwalker:latest
    container_name: earthwalker
    restart: unless-stopped
    ports:
      - "8870:8080"
    environment:
      - EARTHWALKER_GMAPS_API_KEY=YOUR_API_KEY_HERE
      - EARTHWALKER_DB_PATH=/data/earthwalker.db
    volumes:
      - earthwalker_data:/data

volumes:
  earthwalker_data:
docker compose -f ~/services/earthwalker/docker-compose.yml up -d
⚠️
Replace YOUR_API_KEY_HERE with your actual Google API key. Never commit this value to a public repository.
🌐 Step 3 — Caddy Configuration

Add to :

🎮 What Is This?

Earthwalker is a self-hosted alternative to GeoGuessr — the geography game where you're dropped into a random Google Street View location and must identify where in the world you are. You can look around 360°, move along the street, and use any visual clues — road signs, architecture, vegetation, language, climate — to form your best guess.

When you're ready, you place a pin on a world map. Your score is based on how close your pin is to the actual location — up to 5000 points for a pinpoint-accurate guess, dropping off with distance. Multi-round games accumulate scores across several locations.

The self-hosted version supports multiplayer: all players in a room guess the same location simultaneously, with a shared reveal at the end of each round. It's a genuinely great game for groups with varying levels of world geography knowledge.

📖 How to Play
  • Create or join a game — configure the number of rounds and time limit
  • You're dropped into a Street View panorama somewhere on Earth
  • Look around and navigate the streets to gather clues
  • Open the minimap and place your pin where you think you are
  • Submit your guess before the timer runs out
  • See the actual location and your distance — score based on proximity
  • Repeat for all rounds; highest cumulative score wins
⚠️ Known Limitations
IssueDetail
Google API costStreet View API is not free at scale. Monitor your Google Cloud billing. A small friend group playing weekly is unlikely to exceed the free credit.
Coverage gapsGoogle Street View coverage is uneven — some regions have no imagery. Earthwalker draws from the full available pool, so obscure locations do appear.
API key exposureKeep the API key out of version control. It's injected via Docker environment variable — never hardcode it.
👥 Earthwalker supports 1–8 players in a shared round. All players see the same Street View location and guess simultaneously.
🌐 Server Architecture
ComponentDetail
BackendGo HTTP server; manages game rooms and score calculation
DatabaseSQLite — stores games, rounds, and guesses
Street ViewLoaded client-side via Google Maps JavaScript API using random coordinate lookup
Room modelHost creates a room with a shareable link; others join via link
SynchronisationAll players in a room guess the same location; results revealed when all have submitted or timer expires
