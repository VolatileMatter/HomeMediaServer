🐳 Optional — Self-Host a Private Server

Suroi's server component is a Node.js app. For private LAN games:

cd D:\Server
git clone https://github.com/HasangerGames/suroi.git suroi
cd suroi
npm install
# Edit server/src/config.ts for private server settings
npm run build
npm run server

The client then needs to be built pointing at your server address. See the repo README for full private server instructions.

⚠️
Private server setup requires building both the server and client from source. It's non-trivial. For most use cases, linking to the official server is sufficient.

💥 What Is This?

Suroi is a 2D top-down battle royale game — the open-source successor to the beloved but shuttered surviv.io. Players parachute onto an island, scavenge buildings for weapons and healing items, and compete to be the last person alive as a shrinking zone forces players together.

The game is rendered with WebGL for smooth performance in the browser. Weapons include pistols, shotguns, rifles, SMGs, snipers, and melee options. Structures — houses, warehouses, containers — hold loot and provide cover. The zone mechanic creates urgency and funnels engagements.

The HasangerGames team releases regular updates with new weapons, seasonal events (Halloween, Winter, New Year), new structures, and gameplay tweaks. As of v0.30, the game is polished and actively maintained. The wiki at wiki.suroi.io documents weapons, structures, and perks in detail.

🌐 Server Infrastructure

The official Suroi server hosts large-scale public matches. There is no lobby size limit — you're competing in a pool with many players. The game begins when enough players have loaded in.

👥 Private / LAN Play

Private matches require self-hosting both the Node.js game server and a custom-compiled client. This is non-trivial but documented in the GitHub repo. For casual friend groups on Tailscale, the official server is a simpler option — you simply play at the same time and can coordinate via voice chat.
