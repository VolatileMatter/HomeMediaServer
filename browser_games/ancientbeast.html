⚠️ Architecture Note
⚠️
Ancient Beast requires a persistent Node.js server for matchmaking and game state synchronisation. It is not a static file game. Runs as a single Docker container.
🐳 Step 1 — Docker Compose

Create ~/services/ancientbeast/docker-compose.yml:

services:
  ancientbeast:
    image: freezingmoon/ancientbeast:latest
    container_name: ancientbeast
    restart: unless-stopped
    ports:
      - "8840:9000"
docker compose -f ~/services/ancientbeast/docker-compose.yml up -d
💡
Check the GitHub repo for the current Docker Hub image name — it may differ from the above. The project also ships a Dockerfile for building locally if the published image is stale.
🌐 Step 2 — Caddy Configuration

Add to :

ancientbeast.{$DOMAIN} {
  forward_auth  {
    uri /outpost.goauthentik.io/auth/caddy
    copy_headers X-authentik-username X-authentik-groups
  }
  reverse_proxy localhost:8840
}

🎮 What Is This?

Ancient Beast is an open-source turn-based strategy game inspired by Heroes of Might and Magic's combat system. Two players face off on a hex grid battlefield. Each turn, a player summons creatures from their roster by spending plasma energy, then activates those creatures to move and use abilities. The goal is to eliminate all of the opponent's creatures.

Each creature has a distinct set of abilities, movement range, and attack patterns. Effective play requires thinking about ability synergies, controlling key hexes, and managing the plasma economy — spending too much too early leaves you unable to respond to your opponent's plays.

The game has been in community development since 2010 and has grown into a well-balanced tactical experience with a roster of around 15 unique creatures, each with hand-illustrated artwork. It runs entirely in the browser with no client install required.

📖 How to Play
  • One player creates a game room and shares the link with their opponent
  • Each player starts with a pool of plasma energy to spend on summoning creatures
  • Summon a creature by clicking it in the panel — it appears on your side of the grid
  • Each turn, select a creature and choose to move, use an ability, or wait
  • Abilities have cooldowns and plasma costs — manage your energy carefully
  • Eliminate all enemy creatures to win

⚠️ Known Limitations
IssueDetail
1v1 onlyNo AI opponent, no spectator mode, no team matches. Strictly two human players per game.
No save / resumeIf both players disconnect, the game is lost. No mechanism to resume an interrupted match.
Docker image currencyThe published Docker image may lag behind the GitHub repo. Build from source for the latest creature balance patches.
👥 Ancient Beast is a strict 1v1 game. Both players must be online simultaneously — there is no async or correspondence mode.
