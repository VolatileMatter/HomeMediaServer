🎮 What Is This?

Blockrain.js is a clean, minimal Tetris implementation by Aerolab, designed to be embeddable in any web page as a jQuery plugin. On its own it functions as a standalone Tetris game. Multiple visual themes are included — a retro pixel style, a modern minimal look, and others — all swappable via a single config option.

The game implements standard Tetris mechanics faithfully: the seven tetrominoes, wall kicks, line clearing with score multipliers, and escalating speed. An autoplay mode demonstrates the AI solving the board — useful as an ambient display or for testing the theme rendering.

It's the most universally familiar game in this hub — everyone knows Tetris. Having a self-hosted version means no ads, no telemetry, and the same clean experience on any device on the Tailnet.

📖 How to Play
  • Pieces fall from the top — position and rotate them to fill complete horizontal lines
  • Completed lines are cleared and score is awarded — more lines at once scores higher
  • Speed increases over time; the game ends when pieces stack to the top
  • The next piece preview helps plan ahead
Controls
← → Move piece left / right ↑ / Z Rotate piece ↓ Soft drop (faster fall) Space Hard drop (instant placement) P Pause Swipe Move / rotate on touch screens

🏆 Leaderboard

Blockrain tracks a local high score per browser via localStorage. There is no server-side score submission built in.

Tetris-style score is a natural leaderboard candidate. The game exposes a callback on line clear and game over that could post to the hub's score API. Document that work here if undertaken.
