⚠️ Architecture Note
⚠️
Filaments requires a persistent Go server for puzzle storage and multiplayer room management. It is not a static file game. Runs as a single Docker container.
🐳 Step 1 — Docker Compose

Create ~/services/filaments/docker-compose.yml:

services:
  filaments:
    image: ghcr.io/hrfee/filaments:latest
    container_name: filaments
    restart: unless-stopped
    ports:
      - "8830:8080"
    volumes:
      - filaments_data:/data

volumes:
  filaments_data:
docker compose -f ~/services/filaments/docker-compose.yml up -d
💡
The /data volume persists your custom puzzles. Back it up to preserve any puzzles you create.
🌐 Step 2 — Caddy Configuration

Add to :

filaments.{$DOMAIN} {
  forward_auth  {
    uri /outpost.goauthentik.io/auth/caddy
    copy_headers X-authentik-username X-authentik-groups
  }
  reverse_proxy localhost:8830
}

🎮 What Is This?

Filaments is a self-hosted clone of the NYT Strands puzzle — a word search variant where you trace connected paths through a letter grid to find themed words. Unlike a classic word search where letters are independent, Strands requires finding words whose letters form an unbroken chain through adjacent cells, and every letter on the board must be used.

The key differentiator from just playing Strands on the NYT site is twofold: no paywall, and custom puzzle creation. You can design your own themed grids — fill the board with words sharing a theme, define the spangram (the word that spans the board and encapsulates the theme), and share a room link with friends. This makes it an excellent choice for a themed puzzle night — a film series, an inside joke, a hobby topic.

Multiplayer rooms allow everyone to work on the same puzzle simultaneously, seeing each other's progress in real time.

📖 How to Play
  • You're given a letter grid and a theme hint
  • Trace a path through adjacent letters (including diagonals) to spell a themed word
  • Every letter on the board belongs to exactly one word — fill the whole grid
  • Find the spangram — the special word that spans the board and captures the theme — for a bonus
  • In multiplayer rooms, found words are shared with all players in real time
⚠️ Known Limitations
IssueDetail
Custom puzzle creation takes effortBuilding a valid Filaments grid — where all words fit with no leftover letters — requires careful planning. The admin interface helps but it's not instant.
Not the NYT Strands puzzleCustom puzzles only — this does not pull the daily NYT puzzle. For the actual NYT Strands, an NYT subscription is required.
Newer projectLess battle-tested than older games in the hub. Check the GitHub issues before deploying.
👥 Multiplayer rooms sync found words in real time — all players see the board fill up together as words are discovered.

🔧 Server Management
# Check status:
docker compose -f ~/services/filaments/docker-compose.yml ps

# View logs:
docker compose -f ~/services/filaments/docker-compose.yml logs -f

# Update:
docker compose -f ~/services/filaments/docker-compose.yml pull
docker compose -f ~/services/filaments/docker-compose.yml up -d

# Back up custom puzzles:
docker run --rm -v filaments_data:/data -v ~/backups:/backup \
  alpine tar czf /backup/filaments_$(date +%Y%m%d).tar.gz /data
