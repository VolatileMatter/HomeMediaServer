Spelling Bee

// Word Puzzle · Singleplayer · Static Files (Vue 3) · Tailscale + Authentik forward_auth //

A free open-source clone of the NYT Spelling Bee. Given 7 letters arranged in a honeycomb — one centre letter required — find as many words as possible using only those letters. Pangrams (using all 7 letters) score bonus points. Built with Vue 3, runs entirely in the browser.

📁 Step 1 — Clone & Build

Spelling Bee is a Vue 3 project. It requires a one-time build step to produce static files.

cd D:\Server\BrowserGames
git clone https://github.com/ConorSheehan1/spelling-bee.git spelling-bee
cd spelling-bee
npm install
npm run build

The built static files are output to dist/. Serve dist/ as the game folder — update the folder field in index.json accordingly.

💡
Node.js 16+ required for the build. After building, the dist/ folder is fully static — no Node.js needed at runtime. The hub serves it as static files like any other game.

🎮 What Is This?

Spelling Bee is an open-source clone of the New York Times Spelling Bee puzzle — a daily word game that has become one of the most popular casual puzzles on the web. The game presents 7 letters arranged in a hexagonal honeycomb. The centre letter is required in every word; all letters can be reused. The goal is to find as many valid words as possible (minimum 4 letters).

Pangrams — words that use all 7 letters at least once — earn bonus points and are the signature moment of satisfaction in the game. Reaching "Genius" status (top score tier) is the standard challenge.

This implementation by Conor Sheehan uses a built-in word list and puzzle generator, so it functions completely offline after loading. The open-source version generates puzzles independently of the NYT and does not share their specific daily puzzles.

📖 How to Play
  • Click letters on the honeycomb (or type) to spell a word
  • Every word must use the centre letter at least once
  • Words must be at least 4 letters long
  • Letters can be reused as many times as you like
  • Submit with Enter or the Submit button
  • Find a word using all 7 letters — a Pangram — for a bonus score spike
  • Reach the Genius tier (top score threshold) as the daily challenge

🏆 Leaderboard

The game tracks your current session score and found words in localStorage. There is no cross-user leaderboard in the base codebase.

The score tiers (Beginner → Good Start → … → Genius → Queen Bee) function as the built-in progress ladder. Comparing Genius completion times or pangram counts informally works well for a friend group playing the same puzzle.
