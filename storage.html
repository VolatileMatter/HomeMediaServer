Storage & Hard Drives

// Cheap Storage · Internal Migration · No More USB ///

Moving from USB-attached drives to proper internal storage, adding server-grade HDDs affordably, and configuring Windows 10 to use them cleanly.

🐉 V1 Goal: Get your media off USB and onto internal/SATA drives Windows 10 can reliably wake up with. Then expand cheaply with shucked drives.
⚠️ Why USB Drives Fail You
⚡
USB Drives Are Not Made for Always-On Servers USB enclosures spin down aggressively, disconnect during Windows sleep, lose mount letters on reboot, and add latency. For Plex, Minecraft world saves, and seedboxes — they cause corruption and "drive missing" errors.
🖥️ Option A — Transition Your Existing Desktop
🔍 Step 1: Assess What You Have Inside

Open your case and count free SATA ports on the motherboard and free power connectors from the PSU. Most gaming desktops have 4–6 SATA ports. You need one SATA data cable + one SATA power connector per drive.

Device Manager → Disk Drives  (to see currently attached drives)
Disk Management (diskmgmt.msc) → shows all volumes + free slots
💿 Step 2: Buy Cheap Internal SATA Drives

The best bang-for-buck in 2024–2025 is shucked external drives. WD/Seagate externals often contain full enterprise-grade CMR drives and are frequently cheaper than bare drives.

🔧 Step 3: Physical Installation
Power down completely — unplug from wall

Don't just shut down. Fully unplug to discharge capacitors before handling drives.

Seat the drive in a 3.5" bay

Most gaming towers have 2–4 bays. Secure with 4 screws (don't overtighten). If all bays are full, a 5.25" to 3.5" adapter bracket (~$5 on Amazon) lets you use old optical drive bays.

Connect SATA data cable to motherboard + SATA power from PSU

SATA data = flat ribbon cable to any open SATA port on the board. Power = the wider 5-pin SATA power from a PSU cable run.

Boot Windows, open Disk Management (diskmgmt.msc)

New drive appears as "Unknown / Unallocated." Right-click → Initialize (choose MBR for <2TB, GPT for 2TB+). Then right-click unallocated space → New Simple Volume → Format NTFS.

Assign a permanent drive letter

Right-click the volume → Change Drive Letter → pick a high letter like D: or E:. This persists across reboots, unlike USB.

📂 Step 4: Migrate Data Off USB

Use robocopy — Windows' built-in robust file copier that handles permissions and large files correctly:

robocopy "F:\Media" "D:\Media" /E /COPYALL /R:3 /W:5 /LOG:migrate_log.txt

/E = copy all subdirs including empty ones. /COPYALL = preserve timestamps & permissions. /R:3 /W:5 = retry 3 times, wait 5 sec. This is safe to run while files are in use.

💡
After migration: Update your Plex library folder paths in Settings → Libraries → Edit → change path from USB letter to new internal letter. Plex will re-match metadata but won't re-download it.

If you've run out of motherboard SATA ports, a PCIe SATA controller adds 4–8 more ports for ~$20:

🗂️ Windows Storage Spaces — Free RAID-like Pooling

Windows 10 includes Storage Spaces, which lets you pool multiple drives into one volume with optional redundancy — no third-party software required.

Open Storage Spaces

Start → Search "Storage Spaces" → "Create a new pool and storage space"

Select drives to pool

Pick 2+ drives. These will be wiped. Do NOT include your OS drive.

Choose resiliency type

Simple (no redundancy) = most space, no protection. Mirror = 2x copies, needs 2+ drives. Parity = tolerates 1 drive failure, needs 3+ drives. For pure media storage, Simple is fine with backups.

Assign a drive letter and format NTFS

Windows presents the pool as a single large drive. Plex, Minecraft, everything just sees E:\.

⚠️
Storage Spaces ≠ backup. It protects against a single drive dying, but not against ransomware, accidental deletion, or two drives failing at once. Keep a real backup of critical data.
🔧 If You Must Keep USB Drives (Interim Fix)
Disable USB Selective Suspend

Go to Control Panel → Power Options → Change plan settings → Change advanced power settings → USB settings → USB selective suspend setting → set to Disabled.

Also set "Turn off hard disk after" to 0 (Never) in the same advanced settings.

PowerShell (run as admin):
powercfg /change disk-timeout-ac 0
powercfg /change disk-timeout-dc 0
Assign Permanent Drive Letters via Registry

USB drives sometimes reassign letters. Fix by opening Disk Management → right-click each USB volume → "Change Drive Letter and Paths" → Assign a high letter like X: or Z: that won't be taken by other devices. This is stored in the registry and persists across reboots.
