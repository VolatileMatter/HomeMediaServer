🏆 RetroAchievements

Link your RetroAchievements.org account to track achievements in emulated games. Your RA username and token are used client-side only — they're passed to EmulatorJS and never stored on the server.

Get your key at retroachievements.org/settings → Web API Key. This is stored encrypted and only sent to the game emulator.
