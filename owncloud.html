Self-Hosted Cloud Storage

// Nextcloud · Your Own Google Drive · No Docker Needed //

Run your own Google Drive / iCloud replacement on Windows 10. Nextcloud is the recommended choice — more active development than OwnCloud, free, and has Windows desktop and mobile apps.

🐉 V1 Goal: Nextcloud running natively on Windows using XAMPP, accessible on your local network with desktop sync clients on all your devices. SSO via Google OAuth documented in the SSO tab.
⚖️ Nextcloud vs OwnCloud
✅ Nextcloud (Recommended)

Community fork of OwnCloud. More active development, more apps (Calendar, Contacts, Notes, Talk), better mobile apps, and free community edition. Desktop sync client available for Windows, Mac, Linux.

OwnCloud

The original. Solid, enterprise-focused. Slightly simpler setup. The free "Community" edition is good but has fewer apps and slower updates than Nextcloud. Either works fine for personal use.

Recommendation: Go with Nextcloud. Same concept, better apps, more community support. The setup process below applies to Nextcloud.
⚙️ Installation via XAMPP (No Docker) NATIVE WINDOWS
What XAMPP Is

XAMPP is a free Apache + MySQL + PHP bundle for Windows. Nextcloud is a PHP application, so XAMPP gives it everything it needs to run natively. No containers, no Linux subsystem.

Install XAMPP

Download from apachefriends.org. Install to default location C:\xampp. During install, keep at minimum: Apache, MySQL, PHP, phpMyAdmin.

Start Apache and MySQL in XAMPP Control Panel

Open XAMPP Control Panel → click "Start" next to Apache and MySQL. Both should show green "Running."

Create a Nextcloud database

Open http://localhost/phpmyadmin. Click "New" in the left panel. Database name: nextcloud. Collation: utf8mb4_general_ci. Click Create.

Then go to User accounts → Add user account → Username: nextcloud, Host: localhost, set a strong password, check "Grant all privileges on database nextcloud." → Go.

Download Nextcloud

Get the latest zip from nextcloud.com/install. Extract it. You'll get a folder called nextcloud. Move this entire folder to C:\xampp\htdocs\nextcloud.

Configure PHP for Nextcloud

Edit C:\xampp\php\php.ini. Find and change these values:

memory_limit = 512M
upload_max_filesize = 16G
post_max_size = 16G
max_execution_time = 3600
date.timezone = America/Los_Angeles  ; (or your timezone)

Also enable these extensions (remove the semicolon at the start of the line):

extension=intl
extension=gd
extension=exif

Restart Apache in XAMPP Control Panel after saving.

Run the Nextcloud web installer

Navigate to http://localhost/nextcloud. Fill in:

Admin username:   your-admin-name
Admin password:   strong password
Data folder:      D:\NextcloudData   (on your internal drive!)
Database:         MySQL
DB user:          nextcloud
DB password:      (the one you set)
DB name:          nextcloud
DB host:          localhost

Click "Install" — this takes 1–2 minutes.

Run XAMPP Services on Windows Startup

In XAMPP Control Panel → click the checkbox in the "Svc" column next to Apache and MySQL. This registers them as Windows Services that start automatically on boot.

Open firewall for port 80 
netsh advfirewall firewall add rule name="Nextcloud HTTP" dir=in action=allow protocol=TCP localport=80

Access from other devices: http://[your-server-IP]/nextcloud

💻 Desktop & Mobile Sync Clients
💻 Windows Desktop Client

Download from nextcloud.com/install. After install, enter your server URL (http://[IP]/nextcloud) and credentials. It creates a local sync folder that mirrors your cloud storage — just like Google Drive on your desktop.

📱 Mobile Apps

Nextcloud has official iOS and Android apps (free). Connect to your server URL. Enable "Auto Upload" for photos to have your phone's camera roll automatically backed up to your server.

📁 Data Storage Location

During setup you set the data folder to D:\NextcloudData. This is where all uploaded files actually live on disk. The MySQL database just stores metadata. To increase storage, just add more space to that drive (or point to a larger drive — change the path in C:\xampp\htdocs\nextcloud\config\config.php, key datadirectory).

💡
External storage: Nextcloud has an "External Storage" app that lets you mount a local Windows folder as a Nextcloud share — great for exposing your Plex media library through Nextcloud without duplicating files.
🔒 Adding HTTPS (When Ready)

For LAN-only use, HTTP is fine. For remote access, you'll want HTTPS. The easiest path is to put Caddy (a lightweight reverse proxy) in front of XAMPP. Caddy handles free TLS certificates automatically.

# Install Caddy from caddyserver.com
# Create C:\caddy\Caddyfile:

cloud.yourdomain.com {
  reverse_proxy localhost:80
}

# Or for local HTTPS (self-signed):
http://192.168.1.100 {
  tls internal
  reverse_proxy localhost:80
}
🔐 SSO / Google Login → SSO TAB

Nextcloud supports Google OAuth login via the "Social Login" app (install from Nextcloud app store). Users can log in with their Google account. Also supports OIDC, SAML, and LDAP. Full setup walkthrough in the SSO tab.
