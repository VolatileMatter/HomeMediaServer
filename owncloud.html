Nextcloud

// Self-Hosted Google Drive · XAMPP on Windows · FOSS //

Nextcloud is the gold standard for self-hosted cloud storage. Your own Google Drive: file sync, sharing, calendar, contacts, photos — all on hardware you control.

🐉 V1 Goal: Nextcloud running on Windows via XAMPP, accessible on your local network, with desktop sync clients on all your devices. Remote access via Cloudflare Tunnel or Tailscale.
⚙️ Installation on Windows (XAMPP) NO DOCKER
Install XAMPP

Download from apachefriends.org. During install, select: Apache, MySQL, PHP, phpMyAdmin. Install to C:\xampp.

Start Apache and MySQL in XAMPP Control Panel

Open XAMPP Control Panel → click Start next to Apache and MySQL. Check the "Svc" checkbox for both — this makes them Windows Services that auto-start on boot.

Create a Nextcloud database

Open http://localhost/phpmyadmin. Click "New" → database name nextcloud, collation utf8mb4_general_ci → Create.

Go to User accounts → Add user account → username nextcloud, host localhost, set a strong password → check "Grant all privileges on database nextcloud" → Go.

Configure PHP for large file uploads

Edit C:\xampp\php\php.ini:

memory_limit = 512M
upload_max_filesize = 16G
post_max_size = 16G
max_execution_time = 3600
date.timezone = America/Los_Angeles

Enable extensions (remove semicolon):

extension=intl
extension=gd
extension=exif

Restart Apache after saving.

Download and install Nextcloud

Get the zip from nextcloud.com/install. Extract the nextcloud folder to C:\xampp\htdocs\nextcloud.

Run the installer at http://localhost/nextcloud 
Admin username:   your-admin-name
Admin password:   (strong password)
Data folder:      D:\NextcloudData   ← on your big internal drive!
Database:         MySQL
DB user:          nextcloud
DB password:      (from step 3)
DB name:          nextcloud
DB host:          localhost

Click Install — takes 1–2 minutes.

Open firewall 
netsh advfirewall firewall add rule name="Nextcloud" dir=in action=allow protocol=TCP localport=80

Access from other devices: http://[server-ip]/nextcloud

💻 Desktop & Mobile Sync Clients
Windows Desktop Client

Download from nextcloud.com/install. Enter server URL http://[server-ip]/nextcloud. Creates a local sync folder — works identically to Google Drive on the desktop.

Mobile Apps

Official Nextcloud apps for iOS and Android (free). Enable "Auto Upload" for automatic camera roll backup. The files app on iOS can also mount Nextcloud as a Files location.

🌍 Remote Access Options
Option A: Cloudflare Tunnel (Best for Public Access) RECOMMENDED

No port forwarding, no exposed home IP. Cloudflare proxies HTTPS traffic directly to your server through an outbound tunnel.

# Install cloudflared:
winget install Cloudflare.cloudflared

# Authenticate:
cloudflared tunnel login

# Create tunnel:
cloudflared tunnel create nextcloud

# Configure C:\Users\[you]\.cloudflared\config.yml:
tunnel: [your-tunnel-id]
credentials-file: C:\Users\[you]\.cloudflared\[id].json
ingress:
  - hostname: cloud.yourdomain.com
    service: http://localhost:80
  - service: http_status:404

# Run as Windows service:
cloudflared service install
Option B: Tailscale (Best for Private Access)

Install Tailscale on your server and your devices. Access at http://[tailscale-ip]/nextcloud from anywhere. No public exposure.

Option C: HTTPS with Caddy Reverse Proxy
# Caddyfile:
cloud.yourdomain.com {
    reverse_proxy localhost:80
    # Caddy auto-handles TLS via Let's Encrypt
}

Requires port 80/443 open on your router pointing to your server. Your home IP will be in DNS (unless behind Cloudflare proxy).

⚡ Performance Tuning

After initial setup, run the Nextcloud background job (improves performance for large libraries):

# In XAMPP terminal or Windows PowerShell:
cd C:\xampp\htdocs\nextcloud
php occ maintenance:mode --on
php occ db:add-missing-indices
php occ maintenance:mode --off

For large files, consider enabling chunked uploads in the desktop client (Settings → Advanced → Chunked uploads).

🔐 SSO / Google Login → SSO TAB

See the SSO tab for full Google OAuth setup for Nextcloud. Short version: install the "Social Login" app and configure a Custom OpenID Connect provider pointing to Google's OIDC endpoint with your Google OAuth credentials.
