Steam Library

// Server-Stored Games · Sunshine Streaming · Storage Strategy //

Store your Steam library on the server's large drives and stream games remotely via Sunshine+Moonlight. Also covers Steam Link and in-home streaming options.

🐉 V1 Goal: Steam installed on the server desktop with the library on an internal SSD/HDD. Remote play via Sunshine streaming. In-home streaming via Steam Link.
💾 Storage Strategy for Steam Libraries
Steam on a Large Internal Drive

Steam can store its library on any drive letter. Move it to your large internal drive by going to Steam → Settings → Downloads → Steam Library Folders → Add Drive → select your D:\ (or whichever internal drive has space).

When you install a game, select the new library folder. You can also move existing games: right-click game → Manage → Move install folder.

💡
Tiered storage strategy: Keep your 10–20 most-played games on an SSD. Put everything else on a big HDD. Steam lets you have multiple library folders — just install to the appropriate drive per game.

🌐 Remote Play Options
Option 1: Sunshine + Moonlight (Recommended for Best Quality)

See the GameCube tab for full Sunshine setup. For Steam games specifically, add each game as a Sunshine application:

# In Sunshine → Applications → Add:
Name:    [Game Name]
Cmd:     steam://rungameid/[APP_ID]
# Get APP_ID from the game's Steam store page URL

# Or, add Steam itself to launch the launcher:
Name:    Steam
Cmd:     "C:\Program Files (x86)\Steam\Steam.exe"
# Then navigate from the Steam UI

Moonlight users then see all your Steam games listed as Sunshine apps. Controller input is forwarded automatically.

Option 2: Steam Link (Easiest for Same Network)

For in-home streaming only, Steam Link is built into Steam — no extra software needed. On any other PC/phone/TV on your network: install the Steam Link app and pair it to your server. Full 4K60 streaming over gigabit Ethernet.

The Steam Link app is available for Windows, macOS, Android, iOS, Raspberry Pi, and as a dedicated hardware dongle (~$50).

# No configuration needed — just:
# 1. Ensure Steam is running on your server PC
# 2. Open Steam Link on another device on the same network
# 3. Pair using the 4-digit code that appears
Option 3: Steam Remote Play (Built-In, Over Internet)

Steam has built-in remote play via the Steam client. Any friend on your Steam friends list can be invited to play, or you can access your own library remotely via "Steam Remote Play." No port forwarding needed — uses Steam's relay servers.

Quality is lower than Sunshine/Moonlight but requires zero setup. Good fallback when Tailscale/VPN isn't available.

⚙️ Steam Configuration for a Server
Auto-Start Steam in Big Picture Mode

For a server that's primarily accessed via streaming, auto-starting Steam in Big Picture gives a nice gamepad-friendly UI:

# Add to startup (shell:startup shortcut or Task Scheduler):
"C:\Program Files (x86)\Steam\Steam.exe" -bigpicture -start-in-big-picture

# Or run minimized to tray (for background game launching):
"C:\Program Files (x86)\Steam\Steam.exe" -silent
Keeping Games Updated

Steam auto-updates games by default. For a server, you may want to control when updates happen (not mid-session). In Steam settings → Downloads → uncheck "Allow downloads during gameplay." Consider scheduling update windows (e.g. 3–5am when the Minecraft maintenance window is active).

# Force all updates at once from command line:
"C:\Program Files (x86)\Steam\Steam.exe" -forcesteamupdate
Game-Specific Performance Tips for Streaming
  • Run games in borderless windowed mode for best streaming compatibility with Sunshine
  • Set in-game resolution to match your streaming target (1080p60 is usually the sweet spot)
  • Disable VSync in games and use Sunshine's frame pacing instead
  • For CPU-heavy games (strategy, simulation), reduce Sunshine's video encoding quality slightly to free up CPU
🔐 Access Control

Steam accounts are personal — only one account can be logged in at a time. For a server used by multiple people:

  • Your server runs with your Steam account
  • Remote players stream the games you own (via Sunshine) — they're playing on your account
  • For a friend to play their own Steam account remotely, they'd need their own Sunshine setup on their own machine
  • Steam Family Sharing lets others use your library on their own accounts, but not simultaneously with you
