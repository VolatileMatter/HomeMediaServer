⏱️ Step 3 — Play Time Tracking
How tracking works

The Game Library tab tracks play time and last-played per game. Times are stored in browser localStorage. Use the Log Session button on any game card to manually record how long you played. See the Adding Games tab for the optional PowerShell auto-tracking wrapper.

🤖 Automatic Play Time Tracking (Advanced)
PowerShell wrapper — records session time automatically

Instead of pointing Sunshine directly at the game exe, use a wrapper script that records start/stop times to a JSON file.

# D:\Games\GOG\launch_and_track.ps1
param([string]$GameId, [string]$GameExe, [string]$WorkingDir)

$trackFile = "D:\Server\GOGTracker\playtimes.json"
$startTime = Get-Date

Set-Location $WorkingDir
$proc = Start-Process $GameExe -PassThru -WorkingDirectory $WorkingDir
$proc.WaitForExit()

$elapsed = (Get-Date) - $startTime
$minutes = [int]$elapsed.TotalMinutes

$data = if (Test-Path $trackFile) { Get-Content $trackFile | ConvertFrom-Json } else { @{} }
if (-not $data.$GameId) { $data | Add-Member -NotePropertyName $GameId -NotePropertyValue @{totalMinutes=0;lastPlayed=''} }
$data.$GameId.totalMinutes += $minutes
$data.$GameId.lastPlayed = (Get-Date -Format "yyyy-MM-ddTHH:mm:ss")
$data | ConvertTo-Json -Depth 10 | Set-Content $trackFile
# Sunshine command using the wrapper:
powershell -File "D:\Games\GOG\launch_and_track.ps1" ^
  -GameId "divinity-os" ^
  -GameExe "Divinity.exe" ^
  -WorkingDir "D:\Games\GOG\DivinityOriginalSin"

👤 Per-User Save Files

GOG games typically store saves in Documents\My Games\[Game]\ or within the game folder itself. Since only one person streams at a time, saves are naturally isolated for casual use.

For fuller isolation (multiple people playing separate campaigns), the Session Manager symlink approach used for Dolphin and Xenia can be applied: redirect the game's save path via a junction point to D:\Users\[username]\gog\[game]\saves\.
