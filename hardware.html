Server Hardware

// Build · Storage · Networking · Gaming-Ready Servers //

How to build or repurpose a home server that handles media, gaming (Minecraft, ARK, emulation, Steam), and always-on services — at the lowest possible cost.

❓ Your Questions Answered
Can I plug more than one hard drive into a single SATA port?

No — one device per SATA port, natively. Unlike USB, SATA was not designed for daisy-chaining. Each drive needs its own dedicated port and cable.

However, SATA Port Multipliers do exist — they're like SATA "hubs" that let you connect 5 drives to 1 SATA port. The bad news: they're notoriously unreliable, cause performance issues, and many motherboard SATA controllers don't support them. Don't use them for important data.

✅
The right solution for more SATA ports: A PCIe SATA expansion card (~$18–$30) adds 4–8 independent, full-speed SATA ports. Plug it into any PCIe slot. Each port gets its own bandwidth. Fully reliable. This is what NAS builders use.
What is a "shucked" external hard drive?

External hard drives (like a WD Elements 8TB from Best Buy) are sold in a plastic enclosure with a USB bridge board inside. The drive itself is just a standard 3.5" SATA hard drive — identical to drives sold bare.

"Shucking" means opening the plastic enclosure and removing the bare drive to install it directly into your PC over SATA. The reason to do this: external drives are frequently cheaper than bare drives, especially during sales. A 14TB WD Elements on sale for $180 contains the same drive as a $240 bare WD Red Plus 14TB.

How to shuck a WD Elements:
1. Slide a thin plastic pry tool (or folded gift card) into the seam on the short end
2. Pop the clips around all four sides — no screws
3. Slide out the drive + USB board
4. Unplug the USB-to-SATA adapter board from the drive's SATA connector
5. The bare 3.5" SATA drive is ready to install like any other drive

Note: Some WD drives have a "3.3V pin" on the SATA power connector that causes them
to sleep when powered by modern PSUs. Fix: cover pin 3 of the SATA power connector
with a small piece of electrical tape before connecting.
⚠️
Shucking voids the warranty on the external drive — but the warranty on a $20 sale item is worth less than the money you save.
I want to run video games off this server. Does that change the hardware requirements?

Yes, significantly. A pure media server (Plex, Calibre, etc.) can run on nearly anything with 4GB RAM and a slow CPU. But you also want to run:

  • Minecraft server — needs 4–8GB RAM dedicated, modern CPU
  • ARK: Survival Evolved server — needs 8–12GB RAM, strong CPU (ARK is notoriously heavy)
  • Steam library streaming — actual games running on the GPU, full GPU required
  • Dolphin / XENIA emulation streaming — needs a real GPU for rendering at 60fps
  • Sunshine game streaming — needs a GPU with hardware video encoding (NVENC for Nvidia, AMF for AMD)

Bottom line: your server needs to be a real gaming PC, not a NAS or Pi. Your existing gaming desktop is actually ideal — modern CPU, GPU, plenty of RAM. The main upgrade is just internal storage.

🐉
Key insight: Your current gaming desktop already meets these requirements. The main thing missing is enough internal storage. Add SATA drives (see below) and you're good. A "server upgrade" doesn't have to mean new hardware.

How Many Drives Can I Fit?

A typical ATX gaming tower has 2–3 dedicated 3.5" bays. With a 5.25"→3.5" adapter bracket (~$7), you can use old optical drive bays too. Most towers end up with 4–6 drives possible. If you need more:

  • Add a PCIe SATA controller card for more SATA ports
  • An external eSATA enclosure (much better than USB — true SATA over a cable)
  • Or build a separate NAS with a bigger case (Fractal Node 804 = 8 drive bays)

Windows Storage Spaces — Free RAID-like Pooling

Windows 10 includes Storage Spaces. Pool multiple drives into one volume with optional redundancy. Access via Start → "Storage Spaces." Choose Simple (no redundancy, most space) or Mirror (2-drive minimum, survives 1 failure). All your apps see it as a single drive letter.

🌐 Networking for a Home Server
Ethernet, Not WiFi

Your server should be on a wired Ethernet connection. WiFi introduces latency and drops that will affect Minecraft, ARK, game streaming, and Plex 4K streams. If your server is far from your router, a powerline adapter (~$40) runs Ethernet over your house's electrical wiring.

Give Your Server a Static Local IP

Log into your router (usually 192.168.1.1). Find "DHCP Reservations" or "Static IP assignment." Bind your server's MAC address to a fixed IP like 192.168.1.100. This ensures all your port forwards, firewall rules, and bookmarks never break.

📦 Migrating USB Drives to Internal SATA

Use robocopy to safely migrate all data:

robocopy "F:\Media" "D:\Media" /E /COPYALL /R:3 /W:5 /LOG:C:\migrate_log.txt

After migration, update all service configs to point to the new path letters. Then disable USB selective suspend to stop Windows sleeping USB drives if you keep any: Control Panel → Power Options → Advanced → USB selective suspend → Disabled.
