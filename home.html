Physical Server & Network
Hardware, storage, static IP, Ethernet — nothing works without this
START HERE
  • Assess open SATA ports, drive bays, PCIe slots in your gaming desktop
    Count free SATA ports and 3.5" bays. You need room for at least one large HDD (4–8 TB) for media storage alongside the OS SSD.
    30 min
  • Install storage drives and format them NTFS
    Large HDDs for D:\ media library, separate from C:\ OS drive. D:\Users\ for per-user data, D:\Media\ for shared libraries.
    🖧 Hardware →
    1 hr
  • Set a static local IP for the server on your router
    Reserve IP by MAC address in router DHCP settings. Every port rule and service URL depends on this being stable.
    🖧 Hardware →
    15 min
  • Wire the server to the router with Ethernet (not Wi-Fi)
    Streaming, large file transfers, and multi-user access all require the bandwidth and consistency only Ethernet provides.
    15 min
  • Disable sleep and hibernation in Windows Power Settings
    Control Panel → Power Options → Change plan settings → Never sleep. The server must stay awake for remote access.
    🔄 Restarting →
    5 min

🍓 Pi Sentry — Brain of the Server
Raspberry Pi 5 · Docker · Caddy (apt) · Authentik · Everything that doesn't need a GPU
DO THIS SECOND
  • Set up the Pi Sentry — OS, drives, Docker, Tailscale
    Pi OS Lite 64-bit, Penta SATA HAT, mergerfs pool, Docker Compose stack. The Pi is the always-on brain — configure it before touching the PC.
    🍓 Pi Sentry →
    2–4 hrs
  • Deploy Authentik on the Pi via Docker Compose
    Authentik is part of the Pi's main docker-compose.yml. Always on, always available — SSO works even when the PC is off.
    🔐 Authentik SSO →🍓 Pi Sentry →
    45 min
  • Install Caddy natively on the Pi (apt), configure /etc/caddy/Caddyfile
    sudo apt install caddy. Caddy runs as a systemd service — starts automatically on Pi boot. No Docker, no WSL2. All *.ixo.lol subdomains route through here.
    🌐 Network →
    30 min
  • Configure Google OAuth in Authentik, create users, assign groups
    Users sign in with Gmail. No separate passwords to manage. Google handles 2FA.
    🔐 Authentik SSO →
    45 min
  • Create user data folders on Pi (/mnt/archive/Users/[u]/) and PC (D:\Users\[u]\)
    Pi folders for Pi-hosted services. PC folders for desktop emulator saves, MTG decks, AI recommendation history.
    🔐 SSO → User Data Storage →
    15 min

🌐 Tailscale + Caddy — Network & Domain
ixo.lol · Tailscale VPN · Caddy on Pi (apt) · DDNS for Smart TVs
DO THIS THIRD
  • Create Tailscale account, install on server, enable MagicDNS + HTTPS
    Your entire server runs behind Tailscale. No open router ports for any service except Jellyfin. WireGuard-encrypted, free, no bandwidth cap, up to 100 devices.
    🌐 Network →
    30 min
  • Register ixo.lol at Porkbun — set wildcard DNS A record → Tailscale IP
    *.ixo.lol → your Tailscale IP covers all service subdomains. tv.ixo.lol → your home IP for Smart TV Jellyfin. No Cloudflare nameservers needed — Porkbun DNS is fine.
    🌐 Network →
    20 min
  • Set Tailscale ACL rules (hub-and-spoke) in Tailscale admin panel
    Users reach hub only. Hub reaches Pi Sentry and friend nodes. Friend nodes are invisible to users. Paste policy from nodes.json.
    🌐 Network →
    15 min
  • Install Caddy, configure ixo.lol subdomain routing with Authentik forward_auth
    Caddy proxies every *.ixo.lol subdomain to the right local service. Authentik forward_auth sits in front of all of them. One block per service.
    🌐 Network →🛡️ Security →
    1–2 hrs
  • Router: static DHCP for server, port-forward 8096 for Smart TV Jellyfin
    Port 8096 is the ONLY port forwarded to the internet — for LG WebOS, Tizen, Roku, Fire TV apps that cannot run Tailscale. TV apps connect to tv.ixo.lol with Jellyfin native passwords.
    🌐 Network →🎬 Jellyfin →
    15 min
  • Set up Porkbun DDNS script for tv.ixo.lol (home IP may change)
    A scheduled task on the server keeps tv.ixo.lol pointing at your current home IP so Smart TV apps don't break after an ISP IP change.
    🌐 Network →
    15 min
  • Install IPBan, configure Vaultwarden (password manager)
    🛡️ Security →
    30 min
