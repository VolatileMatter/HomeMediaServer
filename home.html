Physical Server & Hardware
Before any software can run, you need a stable machine with proper internal storage
START HERE
  • Decide: use existing gaming desktop or buy a new machine
    Your existing gaming PC covers everything including game streaming. A second server is optional. See the Hardware tab for cost comparison.
    → Hardware tab
    1 hour
  • Assess open SATA ports and drive bays on the case
    Open the case, count free SATA ports on the motherboard and free 3.5" bays. Note how many drives you can add without a PCIe expansion card.
    → Hardware tab
    30 min
  • Buy and install internal storage drives
    Shucked WD Elements externals are the best value (~$15/TB on sale). Install at minimum 2x 14TB for media + 1x SATA SSD for games. Use Storage Spaces to pool them.
    → Hardware tab
    2–4 hours (including shipping wait)
  • Migrate data off USB drives to internal storage
    Use robocopy to copy all media. Update Plex library paths. Remove USB drives or disable their sleep settings if keeping them temporarily.
    → Hardware tab
    2–8 hours (file transfer time)
  • Assign server a static local IP in router
    Log into your router → DHCP reservations → bind the server's MAC to a fixed IP like 192.168.1.100. This prevents all your bookmarks and firewall rules from breaking on reboot.
    15 min
  • Connect server to router via Ethernet cable
    Wi-Fi is not sufficient for a media server. Use a wired connection. If the PC is far from the router, a powerline adapter (~$40) sends Ethernet over house wiring.
    30 min
🐉 Don't skip this phase. Everything else fails on USB drives. Spend the $150–250 on internal storage first — it pays back immediately in reliability.

Security Foundation & Remote Access
Set up the safety net before exposing anything to the internet
BEFORE ANYTHING GOES ONLINE
  • Install Tailscale on server + all your personal devices
    This gives you private remote access to everything without opening any ports. Free. Takes 10 minutes. Do this before any port forwarding.
    → Security tab
    15 min
  • Install Vaultwarden (self-hosted password manager)
    Every service needs a unique password. Set up Vaultwarden first so you have somewhere to store them all securely from day one.
    → Security tab
    30 min
  • Install and configure Caddy reverse proxy
    Caddy handles HTTPS, domain routing, and acts as a security layer in front of all services. Set up a basic Caddyfile even if you don't have a domain yet — you can use LAN IPs.
    → Security tab
    1–2 hours
  • Register a domain (optional but recommended)
    ~$10/yr from Cloudflare Registrar. Enables real HTTPS certs, clean URLs, and Cloudflare proxy to hide your home IP.
    → Domain tab
    30 min
  • Set up SSO (start with Tier 1: per-app passwords)
    Don't try to set up Google OAuth or Authentik yet. Just pick strong unique passwords for each service and store them in Vaultwarden. You'll layer on Google OAuth later.
    → SSO & Users tab
    30 min

🎬 Plex — Priority One
Get your existing media server back online on proper storage
HIGHEST PRIORITY
  • Verify Plex data migrated correctly to new internal drive path
    In Plex: Settings → Libraries → edit each library → update the path from the old USB letter (e.g. F:\) to the new internal drive path (e.g. D:\Media\TV).
    30 min
  • Verify Plex starts automatically on Windows boot
    Plex Media Server installs as a Windows service by default. Check Services (services.msc) → Plex Media Server → set to Automatic. Test by rebooting.
    15 min
  • Invite users to your Plex server via plex.tv
    Settings → Manage Library Access → invite by email. Plex handles its own auth — no SSO needed here.
    15 min
Note: Plex is already installed. This phase is just about ensuring it runs reliably on proper storage. Don't reinstall it — just re-point its libraries.

🌱 Seedbox & Automated Downloads
Mullvad VPN → qBittorrent → Prowlarr → Sonarr → Radarr → Readarr
HIGH PRIORITY
  • Subscribe to Mullvad VPN and install the Windows client
    €5/month, no account name needed. Enable port forwarding. Set to auto-connect on startup with LAN access allowed.
    → Seedbox tab
    30 min
  • Install qBittorrent, bind to Mullvad interface
    The binding is the critical step — it creates an automatic kill switch so downloads stop if VPN drops. Enable the Web UI on port 8090.
    → Seedbox tab
    1 hour
  • Install Prowlarr, Sonarr, Radarr, Readarr
    Prowlarr manages indexers for all the others. Set up in this order: Prowlarr first → connect to Sonarr + Radarr + Readarr → add your TV shows and movies.
    → Seedbox tab
    2–3 hours
  • Connect Sonarr and Radarr to Plex for auto-refresh
    After setup, new downloads appear in Plex within seconds automatically.
    30 min
