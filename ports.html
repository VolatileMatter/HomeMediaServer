⚠️ Resolved Port Collisions
COLLISION FOUND & FIXED: LinguaCafe was originally documented on port 8090 — the same as qBittorrent WebUI. LinguaCafe has been moved to 8093 throughout this documentation. If you already installed LinguaCafe on 8090, change its port in the Docker Compose file and restart.
FIXED: LinguaCafe → now port 8093. Update your docker-compose.yml: change "8090:8090" to "8093:8090" and restart the container.

⚠️ Port Conflict Note — Readeck & Stirling-PDF
OVERLAP RANGE: Readeck per-user instances use ports 8087–8096 by convention. Stirling-PDF is assigned 8090. If you run both, either shift Stirling-PDF to 8097 or limit Readeck to fewer users (8087–8089). Update both the Caddyfile and docker-compose.yml accordingly. The tabs document the recommended ports — adjust to your user count.
